# CountdownPro

一个现代化的活动倒计时页面，具有精美的数字翻转动画效果和多种主题选择。

## ✨ 功能特点

- 📅 **灵活的时间设置** - 自定义目标日期和时间
- ⏱️ **实时倒计时** - 精确显示剩余的天、时、分、秒
- 🎴 **翻转动画** - 炫酷的数字翻转效果，增强视觉吸引力
- 🎨 **多种主题** - 5种精美的渐变色主题可选
- 🌊 **动态背景** - 支持渐变色、粒子动画、波浪动画三种背景效果
- 📱 **响应式设计** - 完美适配手机、平板和桌面设备
- 🎉 **结束特效** - 倒计时结束时显示烟花动画和自定义消息
- 💾 **纯前端实现** - 无需服务器，打开即用

## 🚀 快速开始

### 方法一：直接使用

1. 克隆或下载本项目
2. 在浏览器中打开 `index.html` 文件
3. 设置您的活动信息和目标时间
4. 点击"开始倒计时"按钮

### 方法二：本地服务器

```bash
# 使用 Python 3
python -m http.server 8000

# 或使用 Python 2
python -m SimpleHTTPServer 8000

# 或使用 Node.js
npx serve
```

然后在浏览器中访问 `http://localhost:8000`

## 📖 使用说明

### 基本设置

1. **活动标题** - 输入您的活动名称，如"产品发布会"、"新年倒计时"等

## 🔧 自定义开发

### 修改颜色主题

在 `style.css` 中添加新的主题变量：

```css
[data-theme="yourtheme"] {
    --primary-gradient: linear-gradient(135deg, #color1 0%, #color2 100%);
}
```

### 调整动画速度

在 `script.js` 中修改倒计时更新间隔：

```javascript
// 默认1000ms（1秒）
countdownInterval = setInterval(() => {
    updateCountdown(endMessage);
}, 1000);
```
